Grüß Gott!

Diese App enthält eine Beschreibung meiner wichtigsten Web-Developer Forbildungs-Kurse,
die in meinem öffentlichen GitHub Repository abgelegt sind.

Zweck dieser App

Diese App ist als Übungsprojekt vergangener Kursprojekte gedacht mit dem Ziel,

Praxis in

  1. Angular und Angular Material,
  2. rxJS,
  3. Angular Fire und Web-APIs sowie
  4. Javascript und CSS

zu erhalten.

Diese App soll aber auch meine Kenntnisse in der Webprogrammierung mit Schwerpunkt Angular 9 zeigen.

Diese App enthält daher:

  • Welche Kurse habe ich besucht ⇒ Verzweigen zur lauffähigen App
  • Wie wurden die Projekte technisch umgesetzt ⇒ Verzweigen zum GitHub Repository
  • Wie wurde das Schulungswissen umgesetzt ⇒ diese App mit Source-Code.

Funktionen der App

Anmelden als User oder als Admin (checkbox). Die Login-Daten sind vorbelegt, damit ein einfacher Benutzer-Wechsel möglich ist.

Die Kurse können als Admin verwaltet und als User nur angesehen werden.

Meine Kurse sind geschützt, d.h. diese Kursdaten können nicht geändert werden. Die anderen Kursdaten können geändert, gelöscht und neue Kursdaten angelegt werden.

Kursdaten anzeigen/verwalten nach Login:

  • Darstellung in Tabellenform mit Filter- und Suchfunktion
  • Detail-Informationen des ausgewählten Kurses anzeigen
  • Verzweigen zum Source-Code des Kurses in GitHub.
    Der Kurs wurde nachprogrammiert, wobei die einzelnen Schritte mittels Branches eingecheckt wurden.
  • Verzweigen zur lauffähigen gehosteten App-Version des Kurses

Anzeigen allgemeiner Informationen ohne Login

  • Diese Willkommens-Seite
  • Todo-Liste der App-Entwicklung.
    Die Tabelle kann nach allen Spalten gefiltert werden.
    Die Todos bestehen aus hardcodierten Aktivitäten, die während der Entwicklung erfasst wurden.

Implementierung der App

Verwendeter Stack:

  • Angular 9, Angular Material, Flex-Layout
  • Angularfire (=Google Firebase Datenbank)
  • Unit-Test mit Jasmine & e2e-Test mit Cypress, Doku mit Compodoc
  • Es wurde ein Reactive Design mit rxJS angestrebt.

Tabelle mit Filter- und Sortier-Funktion bei Todo-Liste.

Expansion-Panel und Card bei Kursen

Alle möglichen Funktionen können mittels Button bzw. Menü ausgewählt werden.

Die Kursbestätigung wird in Firebase/Storage gehostet als jpeg-Datei gehostet.

Bei Handies gibt es das Menü als Sidebar und bei größeren Bildschirmen gibt es eine Menüleiste im Header.

Es wurde ein Reactive Design mit rxJS angestrebt.
