Hans Zöchbauers Kurse

Diese App enthält die wichtigsten Kurse die in GitHub abgelegt sind

Geplante Funktionen

  • Speichern der Kursdaten als Admin in Firebase (Zugriff mittels Angular/Fire)
  • Abrufen der Kursdaten und anzeigen als angemeldeteter User mit öffentlichen Credentials. Dadurch kann durch PW-Änderung das Abrufen gesperrt werden
    • Darstellung als mat-table mit Filter- und Suchfunktion
    • Detail-Informationen des ausgewählten Kurses anzeigen
    • Link des Kurses zum GitHub-Repo
    • Link des Kurses zur lauffähigen gehosteten App-Version
  • Anzeigen allgemeiner Infos ohne Login, zB. Todo-Liste mit statischen, d.h. hardcodierten Aktivitäten

Zweck dieser App

  1. Dieses App ist als Übungsprojekt vergangener Kursprojekte gedacht mit dem Ziel Praxis in Angular, Angular Material, rxJS und Angular Fire sowie CSS zu erhalten.
  2. Der Link zu dieser App wird auch in Bewerbungsgesprächen erwähnt, wodurch Firmen die Möglichkeit haben, sich relativ schnell und einfach einen Überblick über meine Kenntnisse in der Web-Programmierung zu machen.
    • Was habe ich an Schulungen gemacht ==> Verzweigen zu lauffähigen App
    • Wie wurden die Projekte technisch umgesetzt ==> Verzweigen zum GitHub Repository
